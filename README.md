# timeConvert

## Usage
Convert 12 clock time to 24 clock

## Dependency
- Python3

## How to run
- goto file directory where the time_convert.py located
- run
```bash
python3 time_convert.py "your 12-clock time here"
```

## Example
```bash
python3 time_convert.py "07:05:45 PM"
```
result:
19:05:45
